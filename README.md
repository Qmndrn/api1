# Скрипт для получения праздников в России через API Calendarific

## Описание

Этот проект представляет собой простой Python-скрипт, который обращается к API Calendarific и получает список российских праздников за указанный год.
Он решает задачу удобного автоматического получения информации о дате, названии и описании праздников.
Используется формат с русскими названиями месяцев, чтобы вывод был понятным пользователю.
